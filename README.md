# ⚙️ TypeScript Projects – Object-Oriented Programming Focus

This repository contains a collection of **TypeScript projects** developed during my second year of the *Web Application Development (DAW)* program.  
Each project focuses on **Object-Oriented Programming (OOP)** principles, using **classes, objects, inheritance, interfaces, and encapsulation** to create modular and reusable code.

---

## 🚀 Overview

Throughout these projects, I explored and implemented:

- Object-oriented design with **classes** and **objects**
- **Inheritance**, **encapsulation**, and **polymorphism**
- **Interfaces** and **abstract classes** for code organization
- Modular architecture and reusable components
- Type safety and static typing provided by TypeScript

These exercises demonstrate my ability to structure clean, maintainable, and scalable TypeScript code using OOP principles.
